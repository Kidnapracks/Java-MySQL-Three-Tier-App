# Java Three-Tier Database Application

This project implements a full three-tier architecture (UI → Business Logic → Data) using JavaFX, JDBC, and MySQL. The application allows users to select any table and column from the Forta database and display the results in a graphical user interface.

---

## Features

- Three-tier architecture:
  - UI Tier: JavaFX interface (FortaUI.java)
  - Business Logic Tier: JDBC operations (DBAccess.java)
  - Data Tier: MySQL database (burnellf)

- Dynamic SQL queries:
  Users can select any table and any column to retrieve live data.

- JDBC connectivity:
  Uses mysql-connector-java to interact with the Forta dataset.

- Clean, modular OOP design:
  Each tier is separated for clarity and maintainability.
